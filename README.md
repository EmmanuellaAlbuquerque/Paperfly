# Paperfly

### How to compile?

```sh
# In paperfly root/folder:

# Linux
~$ find -name "*.java" > sources.txt && javac -d bin @sources.txt
```

### How to run?

```sh
# In paperfly root/folder:
~$ cd ./bin && java view.Main
```
